## What's New

### Avatars Tab Overhaul
- **Avatar Search**:Search public avatars directly from the Avatars tab via avtrdb.com
- **Favorite Groups**:View and manage all six avatar favorite groups with live slot counts (e.g. `Avatars 1 3/25`)
- **Favorite Button**:Favorite or unfavorite any avatar directly from its card; move avatars between groups with one click
- **Platform Badges**:Each avatar card now shows **PC** and/or **Quest** badges indicating which platforms the avatar supports
- **Public / Private Badge**:Moved to the bottom-left of the avatar thumbnail for a cleaner layout
- **VRC+ Indicators**:Favorite groups that require VRC+ are labelled accordingly; slot capacity reflects your subscription (25 or 50 per group)
- Removed the standalone Refresh button from the tab header:the refresh icon next to the search bar already covers this

### Dashboard
- **VRC+ Badge**:A subtle gold-outlined badge appears in the navbar when you have an active VRC+ subscription
