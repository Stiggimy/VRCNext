# 2026.10.7

## Bug Fixes

**Custom Chatbox — Now Playing**
Fixed an issue where the current song title and playback time were not displayed in the chatbox when using the Custom Chatbox feature with the "Now Playing" or "Song Time" variables.

## Requirements

Requires **.NET 9.0 Desktop Runtime (v9.0.14)**. If not installed, you will be prompted to download it automatically when updating.
