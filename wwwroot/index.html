Imports your VRCX history into VRCNext: world time, friend time, instance joins, GPS events, online/offline, status and bio changes. Existing data is preserved and values are added on top. Select your VRCX.sqlite3 file to begin.
warning Experimental Feature

Importing the VRCX database will only add necessary information. This feature is highly experimental and some data may be missing. Instance information works differently in VRCN, so some images or other details might not appear.

If your VRCX database is very important to you, it is recommended to continue using VRCX, as the data will not be fully merged with the VRCN database.

Full instance data will not be available and some elements may not display correctly.

smart_display VRCVideoCacher

Fixes YouTube video errors in VRChat by routing requests through a local proxy. Created by Elly.

The service runs in the background. When you close VRCNext, the process will be terminated.

If you are running this service for the first time, please follow the setup instructions that appear when starting it. Also make sure to read the usage guide on the GitHub page: github.com/EllyVR/VRCVideoCacher

How to use YouTube Fix?
1 Click the Install / Update button above to install the YouTube Fix patcher.
2 Make sure you create a throwaway YouTube account.
3 Open Google Chrome or Mozilla Firefox and log into your throwaway YouTube account.
4 Click Install Chrome Extension or Install Firefox Extension above, you need the browser extension to make YouTube Fix work.
5 Once the browser extension is installed, make sure to start the YouTube Fix service using the Start button above.
6 Refresh the YouTube page until the YouTube Fix (VRCVideoCacher) shows "Valid Cookies".
